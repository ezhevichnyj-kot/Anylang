# Anylang
## Pet-проект по теме: "Перевод научных статей"

### Команда разработки:

- ezhevichnyj-kot - Back-end, работа с API
- SanyaPulemet - Front-end, дизайн

### Стек технологий:

- Django
- HTML + CSS + Bootrstrap
- API Deepl

- Загрузка PDF файлов
- Перевод файлов на различные языки
